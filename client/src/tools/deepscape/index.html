Deepscape ⚖️

Pannable chartspace with editor nodes for visual data flows

Overview

Deepscape is an Elixir-based data visualization and flow editor that allows users to create interactive, node-based data processing pipelines with an infinite canvas approach.

Key Features

  • ✓
    Infinite Canvas

    Pan and zoom through an unlimited workspace for visual data flows

  • ✓
    Node-Based Editing

    Create data pipelines by connecting nodes with visual wires

  • ✓
    Real-Time Collaboration

    Built on Elixir for multi-user editing and viewing

  • ✓
    Data Visualization

    Integrated charts and graphs for immediate data insights

Development Status

Deepscape is currently at medium speed development status. The core architecture is functional, but there are performance optimizations still in progress for larger datasets.

WebSocket connectivity issues are currently being addressed to improve the real-time collaboration features.
