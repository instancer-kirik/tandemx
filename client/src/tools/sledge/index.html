Sledge ⚡

Web browser with anti-flashbang protection and better tab management

Overview

Sledge is a PyQt-based web browser designed to solve common annoyances with standard browsers, particularly focusing on protecting users from sudden bright flashes and improving tab management.

Key Features

  • ✓
    Anti-Flashbang Protection

    Automatically detects and dims sudden bright screen changes to protect your eyes

  • ✓
    Enhanced Tab Management

    Hierarchical tab organization and intelligent grouping

  • ✓
    Resource Management

    Intelligently pauses unused tabs to save system resources

  • ✓
    PyQt Implementation

    Built with Python and Qt for cross-platform compatibility

Development Status

Sledge is currently in active development. Some features are working in prototype form, while others are still being implemented.

The browser uses a QtWebEngine backend with custom Python handlers for the special features.
