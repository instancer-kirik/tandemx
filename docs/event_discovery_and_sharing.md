## Schedule Sharing Features

### Schedule Sharing with Friends

```
+----------------------------------------------------------------------+
|  TandemX                                        [User: Artist Name]  |
+----------------------------------------------------------------------+
| [Home] [Calendar] [Discover] [Applications] [Messages] [Profile]      |
+----------------------------------------------------------------------+
|                                                                      |
|  SHARE SCHEDULE                                                      |
|                                                                      |
|  Select Date Range:                                                  |
|  Start: [07/15/2023]    End: [07/17/2023]                            |
|                                                                      |
|  Select Events to Share:                                             |
|  [x] Summer Festival - July 15-17, 2023                              |
|  [x] Studio Session - July 16, 2023, 10:00 AM - 12:00 PM             |
|  [ ] Private Meeting - July 16, 2023, 2:00 PM - 3:00 PM              |
|  [x] Afterparty - July 16, 2023, 11:00 PM - 2:00 AM                  |
|                                                                      |
|  Share With:                                                         |
|  [x] Friend Name                                                     |
|  [ ] Friend Name                                                     |
|  [x] Friend Name                                                     |
|  [+ Select More]                                                     |
|                                                                      |
|  Visibility Options:                                                 |
|  (o) Show event details                                              |
|  ( ) Show busy/free status only                                      |
|  ( ) Show event titles only                                          |
|                                                                      |
|  Add Message:                                                        |
|  +------------------------------------------------------------------+|
|  | Here's my schedule for the festival weekend. Let me know when    ||
|  | you'll be there so we can meet up!                               ||
|  +------------------------------------------------------------------+|
|                                                                      |
|  [Cancel]                                        [Share Schedule]    |
+----------------------------------------------------------------------+
```
